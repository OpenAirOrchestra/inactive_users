Lists users that are not active on the site.

== Description ==

Lists users that have not logged in in a given period, and whose accounts are
of a minimum age.


== Installation ==

1. Install the excellent Login Logger plugin by Stephen Merriman
2. Upload this plugin to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Why did you create this plugin =

The members site for Carnival Band users was getting clogged with inactive
accounts.
